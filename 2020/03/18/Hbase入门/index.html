Hbase入门

一、Hbase简介

​ Hbase是一个开源、分布式、可扩展、面向列的高性能非关系型数据库

  1. Hbase支持随机、实时的大数据读写
  2. 基于HDFS，实现数据容错和高可用，可以实现横向扩展
  3. 支持上百亿行，百万列的数据存储

二、Hbase和RDBMS的区别

Hbase RDBMS
硬件架构 基于HDFS，硬件成本低廉 传统的多核系统，硬件成本昂贵
容错性 基于HDFS实现高可用 一般需要额外硬件设备实现 HA 机制
数据库大小 PB GB、TB
数据类型 byte\string 丰富的数据类型
事务支持 ACID只支持单个row级别 全面的 ACID 支持，对 Row 和表
查询语言 只支持java API SQL
索引 只支持row-key 支持
吞吐量 百万查询/每秒 数千查询/每秒
数据操作 增删改查，不支持join 各种各样的函数与表连接
存储模式 列式存储 基于表结构和行式存储

三、Hbase的数据模型

数据在逻辑上的排布：

Key Value(CF,Column,Version)
1 info{‘姓’: ‘张’，’名’:’三’}
pwd{‘密码’: ‘111’}
2 Info{‘姓’: ‘李’，’名’:’四’}
pwd{‘密码’: ‘222’}
